# Справочная Коломны — Telegram-бот

AI-помощник для жителей городского округа Коломна Московской области. Консультирует по земельно-имущественным отношениям, нестационарной торговле, государственным услугам и принимает официальные обращения.

## Возможности

- Ответы на вопросы по ЗК РФ, НТО, расселению, РПГУ МО
- Веб-поиск для актуальных ответов (встроен в Qwen)
- Форма официального обращения (ФИО, телефон, тема, суть)
- Пересылка обращений в Telegram-группу сотрудников администрации

---

## Шаг 1. Создать ботов в Telegram

1. Откройте Telegram, найдите `@BotFather`
2. Отправьте `/newbot`
3. Придумайте имя (например: `Справочная Коломны`) и username (`kolomna_info_bot`)
4. Скопируйте токен — это `CITIZEN_BOT_TOKEN`
5. Повторите для второго бота (например: `kolomna_admin_bot`) — это `ADMIN_BOT_TOKEN`

## Шаг 2. Настроить группу для сотрудников

1. Создайте группу в Telegram для сотрудников администрации
2. Добавьте в эту группу `@kolomna_admin_bot` (второй бот)
3. Узнайте `chat_id` группы:
   - Добавьте в группу бота `@userinfobot`
   - Он пришлёт сообщение с chat_id (отрицательное число, например: `-1001234567890`)
   - Это и есть `ADMIN_GROUP_CHAT_ID`
4. Удалите `@userinfobot` из группы

## Шаг 3. Получить API-ключ Qwen

1. Зайдите на [dashscope.aliyuncs.com](https://dashscope.aliyuncs.com)
2. Зарегистрируйтесь (бесплатно, нужна почта)
3. Перейдите в раздел `API Keys` → создайте ключ
4. Скопируйте ключ — это `DASHSCOPE_API_KEY`

> **Бесплатный тариф:** 1 000 000 токенов в месяц для новых пользователей — при небольшой нагрузке хватит надолго.

## Шаг 4. Настроить конфигурацию

```bash
cp .env.example .env
```

Откройте `.env` и заполните все значения:

```
CITIZEN_BOT_TOKEN=7123456789:AAF...
ADMIN_BOT_TOKEN=7987654321:AAG...
ADMIN_GROUP_CHAT_ID=-1001234567890
DASHSCOPE_API_KEY=sk-...
```

## Шаг 5. Запуск локально (для теста)

```bash
pip install -r requirements.txt
python main.py
```

## Деплой на Koyeb (бесплатно навсегда, без карты)

### 1. Создать GitHub-репозиторий
```bash
git init
git add .
git commit -m "Initial commit"
# Создайте репо на github.com и запушьте
git remote add origin https://github.com/YOUR_USERNAME/kolomna-bot.git
git push -u origin main
```

### 2. Задеплоить на Koyeb
1. Зарегистрируйтесь на [koyeb.com](https://koyeb.com) (через GitHub или email — без карты)
2. Нажмите **Create App** → **Web Service**
3. Выберите **GitHub** → укажите ваш репозиторий
4. Koyeb автоматически обнаружит `Dockerfile`
5. Перейдите в **Environment Variables** и добавьте все переменные из `.env`:
   - `CITIZEN_BOT_TOKEN`
   - `ADMIN_BOT_TOKEN`
   - `ADMIN_GROUP_CHAT_ID`
   - `DASHSCOPE_API_KEY`
   - `DASHSCOPE_MODEL` = `qwen-plus`
6. Нажмите **Deploy**

После деплоя бот будет работать 24/7 без засыпания. При каждом `git push` Koyeb автоматически перезапустит бота.

## Структура проекта

```
├── main.py              # Точка входа
├── requirements.txt     # Зависимости Python
├── Dockerfile           # Контейнер для Koyeb
├── .env.example         # Шаблон конфигурации
└── src/
    ├── config.py        # Загрузка переменных окружения
    ├── knowledge_base.py # Системный промпт (правовая база)
    ├── ai_engine.py     # Интеграция с Qwen API
    ├── state_manager.py # Хранение состояния диалога (SQLite)
    ├── appeals.py       # Форма обращения и отправка в группу
    └── bot_handlers.py  # Telegram-хендлеры
```

## Команды бота

| Команда | Действие |
|---------|----------|
| `/start` | Приветствие и запуск |
| `/help` | Справка и контакты |
| `/reset` или `/новый` | Сбросить историю диалога |
| 📨 Оставить обращение | Кнопка для подачи официального обращения |
